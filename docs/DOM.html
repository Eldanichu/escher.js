Class: DOM

DOM(parentDOM, type)

new DOM(parentDOM, type)

A DOM object transformed using CSS3D to ver included in the graph. DOM objects always stay on top of everything else, mouse events are not supported for these. Use the normal DOM events for interaction.
Parameters:
Name Type Description
parentDOM Parent DOM element that contains the drawing canvas.
type Type of the DOM element (e.g. "div", "p", ...)
