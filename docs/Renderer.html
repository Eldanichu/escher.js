Methods

update(object, viewport)

Update the renderer state, update the input handlers, calculate the object and viewport transformation matrices. Render the object using the viewport into a canvas element. The canvas state is saved and restored for each individual object, ensuring that the code of one object does not affect another one. Should be called at a fixed rate preferably using the requestAnimationFrame() method.
Parameters:
Name Type Description
object Object to be updated.
viewport Viewport to be updated (should be the one where the objects will be rendered after).
