Class: Node

Node()

new Node()

Node objects can be connected between them to create graphs. Each node contains inputs, outputs and a set of attributes containing their state. Inputs can be connected to outputs of other nodes, and vice-versa. This class implements node basic functionality, the logic to connect node and define inputs/outputs of the nodes.

registerSockets

This method should be used for the node to register their socket inputs/outputs. It is called automatically after the node is added to the node graph to create sockets.
